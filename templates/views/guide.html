Admin Guide: How to Run and Close the System

How to Run the System

Follow these steps to activate and run your system in the command prompt:

  1. Navigate to your project folder:
    2. cd C:\new-facial-recognition
  2. Activate the virtual environment:
    3. Type with double quotation marks ".venv\Scripts\activate"
  3. Once activated, run the system
    4. Type "flask run" or "python app.py"

How to Close the System

Follow these steps to deactivate the virtual environment and close the system:

  1. To stop the system or server, press Ctrl + C in the command prompt.
  2. To deactivate the virtual environment:
    3. Type "deactivate"
